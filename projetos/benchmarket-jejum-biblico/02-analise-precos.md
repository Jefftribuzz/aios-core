## 💭 PSICOLOGIA DE PREÇO

### Gatilhos Identificados

**Preço Truncado (anchor bias)**
```
"De R$ 999 por apenas R$ 297"  ← Percepção: 70% desconto
"3x R$ 99 = R$ 297"             ← Parcelamento invisível
"Hoje: R$ 197 | Amanhã: R$ 297" ← Urgência
```

**Preço Premium (percepção de valor)**
```
"Investimento de R$ 997"        ← "Investimento" vs "Gasto"
"Mentoría para apenas 15 pessoas" ← Exclusividade
"R$ 500/pessoa vs R$ 3.000 group" ← Valor comparativo
```

**Free/Paid Ratio**
```
Lead magnet: R$ 0
Mini-tripwire: R$ 47
Main offer: R$ 297 (multiplicador 6x)
Premium: R$ 997 (multiplicador 3x de main)
```

## ⚠️ OBSERVAÇÕES IMPORTANTES
