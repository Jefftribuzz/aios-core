# 📱 Instruções para Criar o Carrossel no Canva

## Pré-requisitos
- Conta Canva (grátis ou Pro)
- Acesso a este documento com as especificações

---

## Passo a Passo

### 1. Criar o Novo Design

1. Acesse [canva.com](https://www.canva.com)
2. Clique em **"Create a design"**
3. Busque por **"Instagram Carousel Post"** ou **"Carousel"**
4. Canva automaticamente criará com as dimensões corretas (1080x1350px por slide)

### 2. Configurar o Template Base

#### Slide 1 - CAPA
```
Texto Principal (em verde):
"Vc acha que um Curso Online 
deve ser criado primeiro que uma Mentoria?"

Estilo:
- Font: Montserrat Bold ou similar (SEM SERIFAS)
- Tamanho: 56-64px
- Cor: Verde (#4CAF50)
- Fundo: Preto sólido (#000000)
- Alinhamento: Centro
- Espaçamento: Generoso (top 200px, bottom 200px)
```

---

#### Slide 2 - CARD 1 (Métricas)
```
Pergunta:
"Quantas vendas de 97 reais de um infoproduto 
precisa fazer para fatura 10 mil reais?"

Subtítulo:
"...ou seja, 1 venda de mentoria para quem 
está começando?"

Estilo:
- Título: 48px, Branco, Montserrat Bold
- Subtítulo: 36px, Verde (#4CAF50), negrito
- Fundo: Preto
- Alinhamento: Centro com padding 100px top/bottom
- Ícone opcional: Símbolo de moeda/calculadora em verde
```

---

#### Slide 3 - CARD 2 (Beneficiários)
```
Título:
"Com a mentoria você gera caixa para 
reinvestir e crescer com estrutura sem precisar:"

Lista de Benefícios (em verde):
"✓ Fazer lançamentos
✓ Tráfego pago
✓ Muitos conteúdos"

Estilo:
- Título: 40px, Branco, Montserrat Bold
- Lista: 32px, Verde (#4CAF50), Montserrat Bold
- Fundo: Preto
- Espaçamento: Generoso
- Use checkmarks (✓) em verde
```

---

#### Slide 4 - CARD 3 (Metodologia)
```
Texto Principal:
"Existe uma metodologia que estrutura os 
entregáveis da sua mentoria de forma dinâmica 
por um ano"

Destaque:
"que vai atrair clientes que pagam +10 mil 
reais pela sua mentoria - e eles ainda 
acham barato."

Estilo:
- Texto: 44px, Branco, Montserrat Bold
- Destaque: 48px, Verde (#4CAF50), Montserrat Bold
- Fundo: Preto
- Alinhamento: Centro
- Padding: 150px top/bottom
```

---

#### Slide 5 - CTA (Call-to-Action)
```
Chamada Principal:
"Comente Mentoria"

Descrição:
"e ganhe uma Raio X da sua ideia ou 
modelo de mentoria atual para conseguir 
clientes de Hight Ticket"

Estilo:
- Principal: 56px, Verde (#4CAF50), Montserrat Bold
- Descrição: 32px, Branco, Montserrat Regular
- Fundo: Preto
- Alinhamento: Centro
- Adicionar seta ou ícone em verde apontando para baixo
```

## Checklist Visual

### Paleta de Cores ✓
- [ ] Fundo: Preto (#000000)
- [ ] Texto Principal: Branco (#FFFFFF)
- [ ] Destaque: Verde (#4CAF50)
- [ ] Sem outras cores além dessas 3

### Typography ✓
- [ ] Font consistente em todos os slides (Montserrat ou similar)
- [ ] Sem serifas
- [ ] Tamanhos hierárquicos (Títulos 48-64px, Corpo 32-40px)
- [ ] Espaçamento entre linhas generoso (1.6-1.8)

### Layout ✓
- [ ] 5 slides no total
- [ ] Slide 1: Capa/Hook
- [ ] Slides 2-4: Conteúdo (Cards)
- [ ] Slide 5: CTA
- [ ] Alinhamento consistente (centro/esquerda/direita - escolher um)

### Copy ✓
- [ ] Todas as palavras-chave em verde
- [ ] Sem erros de digitação
- [ ] Copy concisa e impactante
- [ ] CTA claro no último slide

### Espaçamento ✓
- [ ] Padding: 100-150px nas laterais
- [ ] Padding: 100-200px top/bottom
- [ ] Espaço branco abundante
- [ ] Sem textos muito juntos

## Salvando e Compartilhando

### Salvar no Canva
1. Clique em **"Save"** ou o Canva salva automaticamente
2. Nomeie como: **"Social Media Jeff - Mentoria vs Curso (v1)"**

### Compartilhar com a Equipe
1. Clique em **"Share"** (canto superior direito)
2. Selecione **"Copy link"** ou convide via email
3. **IMPORTANTE**: Deixar com permissão de **"Edit"** (não apenas view)
4. Cole o link em [LINKS-CANVA.md](./LINKS-CANVA.md)

### Publicação no Instagram
1. Clique em **"Download"**
2. Formato: **MP4** (para melhor visualização) ou **PNG** individual
3. Tamanho: Deixar padrão do Canva
4. Salvar e fazer upload no Instagram

## Dicas Profissionais 🎨

1. **Contraste**: Certifique-se que o branco é bem visível no preto
2. **Teste Mobile**: Visualize como ficaria em um celular (1080x1350 é a resolução mobile)
3. **Tipografia**: Evite fontes muito finas ou muito espessas
4. **Espaçamento**: Quanto maior o espaço branco, melhor a legibilidade
5. **Cor Verde**: Use verde natural, não neon (#4CAF50 é bom, mas teste)
6. **Call-to-Action**: Torne super claro o que o usuário deve fazer
7. **Ícones**: Use ícones simples e limpos (Canva tem muitos grátis)
