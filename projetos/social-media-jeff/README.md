## Design Guidelines

### Paleta de Cores
```
Fundo:    #000000 (Preto)
Texto:    #FFFFFF (Branco)
Accent:   #4CAF50 (Verde) - Palavras-chave
```

### Typography
- **Título/Pergunta**: Font grande, ousada, branca
- **Texto de Destaque**: Verde, tamanho 1.5x do corpo
- **Corpo**: Branco, espaçamento generoso, máximo 3-4 linhas por slide

### Elementos Visual
- Espaço branco generoso
- Ícones simples em verde ou branco
- Sem excesso de elementos gráficos
- Foco na legibilidade

## Como Criar/Editar no Canva

### Para Novos Carrosseis
1. Acesse [canva.com](https://www.canva.com)
2. Crie novo design: "Instagram Carousel Post" (1080x1350px)
3. Aplique paleta de cores padrão do projeto
4. Siga o template visual definido
5. Compartilhe o link editável com a equipe

### Checklist de Design
- [ ] 5 slides (Capa, 3 Cards, CTA)
- [ ] Todas as palavras-chave em verde
- [ ] CTA claro no último slide
- [ ] Contraste suficiente (branco/preto)
- [ ] Tipografia consistente
- [ ] Sem elementos que saiam do padrão
