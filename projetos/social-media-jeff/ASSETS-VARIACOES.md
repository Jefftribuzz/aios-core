## 📐 Dimensões & Formatos

### Instagram Carousel
- **Dimensão por slide**: 1080px Width × 1350px Height
- **Formato**: 4:5 (recomendado para Stories também)
- **Arquivo**: MP4 (preferido) ou PNG (alternativa)

### Tamanhos de Fonte (Recomendados)
```
Título Principal:         56-64px (Montserrat Bold)
Subtítulo/Destaque:       40-48px (Montserrat Bold)
Corpo/Texto secundário:   32-36px (Montserrat Regular)
Pequeno (nota):           24-28px (Montserrat Regular)
```

### Espaçamento
```
Padding horizontal:       100-150px
Padding vertical (top):   150-200px
Padding vertical (bottom): 150-200px
Line height:              1.6 - 1.8
Spacing between elements: 30-50px
```

## 🔤 Variações de Tipografia

| Peso | Use Para | Exemplo |
|------|----------|---------|
| **Light (300)** | Subtítulos pequenos | "Slide 2 de 5" |
| **Regular (400)** | Corpo/texto | Descrições, benefícios |
| **Medium (500)** | Ênfase leve | Palavras-chave |
| **Bold (700)** | Títulos, destaque | Perguntas principais |
| **Black (900)** | Máximo impacto | Números, metrics |

**Fonte Recomendada**: Montserrat (grátis no Google Fonts)  
**Alternativas**: Poppins, Inter, Roboto (todas sem serifas)

## 🎬 Versões para Diferentes Plataformas

### Instagram Feed (Carousel)
- ✅ 1080x1350 (padrão)
- ✅ Duração: ilimitada (são imagens)
- ✅ Textos legíveis em mobile

- Duração: 15-90s
- Dimensão: 1080x1920
- Adicione som/música

### TikTok (futura adaptação)
- Dimensão: 1080x1920 (9:16)
- Duração: 15-60s
- CTA mais direto

### Pinterest (futura adaptação)
- Dimensão: 1000x1500 (2:3)
- Foco em texto + design limpo
